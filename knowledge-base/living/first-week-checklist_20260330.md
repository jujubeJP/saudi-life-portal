# サウジアラビア到着後 最初の1週間チェックリスト

## Day 1: 到着日
- [ ] 空港でSIMカード購入（STC推奨、パスポート持参）
- [ ] Uber / Careem アプリをインストール
- [ ] ホテル or 住居にチェックイン
- [ ] 現金の確保（ATM or 両替所）

## Day 2-3: 基本セットアップ
- [ ] 会社への出社・手続き開始
- [ ] イカーマ（在留許可証）の申請手続き確認（スポンサー主導）
- [ ] 住居の確認（コンパウンド or アパートメント）
- [ ] 周辺のスーパー・レストランの確認

## Day 4-5: デジタルセットアップ
- [ ] Absher アプリのダウンロード・登録（イカーマ番号必要）
- [ ] Tawakkalna アプリのダウンロード・登録
- [ ] Nafath（国民認証）の登録
- [ ] HungerStation / Jahez（フードデリバリー）インストール

## Week 2: 生活基盤
- [ ] 銀行口座開設（イカーマ取得後）
- [ ] 光熱費契約（SEC電気、NWC水道）
- [ ] 在留届の提出（在サウジ日本大使館）
- [ ] 医療保険の確認（通常は雇用主が手配）

## 優先度の判断基準
- **イカーマなしでできること**: SIM購入、Uber/Careem、現金引き出し、在留届
- **イカーマが必要なもの**: 銀行口座、Absher登録、運転免許、住居契約
- **銀行口座が必要なもの**: STCpay登録、mada Pay、ポストペイドSIM

## よくある落とし穴
1. イカーマの発行が遅れる（2-4週間かかることも）→ その間は現金+クレカで乗り切る
2. 帯同家族の出国許可を取り忘れる → Absherで事前申請必須
3. 常備薬の英文処方箋を持ってこなかった → 没収リスク
4. 電源プラグがType G（イギリス式）で日本のプラグが使えない
